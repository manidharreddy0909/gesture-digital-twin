SUPPORT & TROUBLESHOOTING

Issue: "PyQt6 not installed" error
Solution: pip install PyQt6 PyQt6-Charts

Issue: Low FPS (<30)
Solution:
  - Disable ML gestures: enable_ml_gestures=False
  - Use exponential smoothing: use_kalman_filter=False
  - Check profiling for bottleneck

Issue: Cursor jumps/jerky
Solution:
  - Increase smoothing_factor (0.3-0.5)
  - Enable adaptive_smoothing=True
  - Use Kalman filter: use_kalman_filter=True

Issue: Two-hand cursor goes backwards
Solution:
  - This has been fixed in hand_tracker.py (lines 214-232)
  - Run debug_hands.py to verify
  - Check that mirror flip happens before hand_id assignment

Issue: Gestures not detected
Solution:
  - Adjust threshold values in config.py
  - Run calibration (press 'C')
  - Increase motion_history_size for better tracking

Issue: Integration not connecting
Solution:
  - Verify backend server is running
  - Check endpoint URL/port in config
  - Run with enable_profiling=True to see connection logs
  - Check backend logs for errors

Issue: High latency (>50ms)
Solution:
  - Use exponential smoothing instead of Kalman
  - Reduce resolution or frame rate
  - Disable unnecessary features
  - Profile to find bottleneck
